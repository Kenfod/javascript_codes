String slice

        The slice() method in JavaScript is used to extract a portion
         of a string and create a new string without modifying the original string.

        Syntax: string.slice(startingIndex, endingIndex);
        
        Parameters:
        This method uses two parameters. 
        This method does not change the original string.
        A negative number selects from the end of the string.

        startingIndex: It is the start position and it is required(The first character is 0).

        endingIndex: (Optional)It is the end position (up to, but not including). 
        The default is string length.

        Return Values:
        It returns a part or a slice of the given input string.
