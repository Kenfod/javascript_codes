Constants

        JavaScript constants are the variables whose values remain unchanged throughout the execution of the program. 
        You can declare constants using the const keyword.


        The const keyword was introduced in the ES6 version of JavaScript with the let keyword. 
        The const keyword is used to define the variables having constant reference.

        A variable defined with const can't be re-declared, reassigned. The const declaration have block as well as function scope.

        You always need to assign a value at the time of declaration if the variable is declared using the const keyword.
