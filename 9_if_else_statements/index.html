if-else Statements

        JavaScript if-else statement executes a block of code based on a condition. 
        If the condition evaluates to true, the code inside the "if" block executes;
        otherwise, the code inside the "else" block, if present, executes.

        Such control statements are used to cause the flow of execution 
        to advance and branch based on changes to the state of a program.
