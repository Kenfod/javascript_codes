reduce() Method

      The JavaScript Array.reduce() method iterates over an array, 
      applying a reducer function to each element, 
      accumulating a single output value. 
      
      It takes an initial value and processes elements from left to right, 
      reducing the array to a single result. 
      
      It is useful for doing operations like max in an array, 
      min in an array and sum of array
